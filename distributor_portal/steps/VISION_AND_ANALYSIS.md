## 🎯 المفهوم

**Distributor Portal** = منصة B2B للموزعين للبيع للبقالات

### الهدف:
```
Distributors can:
├── Manage product catalogs
├── Set wholesale prices
├── Create bulk offers
├── Receive orders online
├── Track deliveries

```
