## Design Guidelines

- **Resolution:** 1920x1080 (desktop screenshots)
- **Browser:** Chrome with minimal UI (hide bookmarks bar)
- **Language:** Arabic (RTL)
- **Data:** Use demo/test data only — no real business data
- **Brand:** Use Alhai brand colors for any overlay text
- **Format:** PNG, high-quality
