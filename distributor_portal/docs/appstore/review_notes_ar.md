# ملاحظات — بوابة الموزّعين

## ملاحظات ما قبل الإطلاق

### متطلبات الإطلاق

1. **النطاق (Domain):**
   - تسجيل وتوجيه `portal.alhai.store` لـ Netlify
   - شهادة SSL مفعّلة (تلقائي مع Netlify)

2. **البيئة:**
   - Supabase production project مُهيّأ
   - Sentry project مُنشأ بـ DSN حقيقي
   - متغيرات البيئة مكتملة في Netlify

3. **الأمان:**
   - MFA مُفعّل وتم اختباره لحسابات super_admin
   - CSP headers مُعدّة في index.html
   - RLS policies مُراجعة ومُختبرة

4. **المحتوى:**
   - سياسة الخصوصية منشورة ومربوطة
   - شروط الخدمة منشورة ومربوطة
   - manifest.json مُحدّث بالاسم العربي

### حسابات تجريبية

```
الموزّع:
- البريد: demo-distributor@alhai.store
- الرقم السري: [يُنشأ قبل الإطلاق]

المشرف (super_admin):
- البريد: demo-admin@alhai.store
- الرقم السري: [يُنشأ قبل الإطلاق]
- MFA: [يُفعّل عند أول تسجيل دخول]
```

### سيناريوهات الاختبار

1. **تسجيل الدخول:** بريد + كلمة مرور → لوحة التحكّم
2. **MFA:** تسجيل دخول super_admin → تحقق TOTP → لوحة التحكّم
3. **المنتجات:** إضافة منتج → تعديل سعر → حذف منتج
4. **الفوترة:** إنشاء فاتورة → عرض QR code → طباعة A4
5. **الطلبات:** عرض طلبات واردة → تحديث الحالة → إتمام
6. **الإعدادات:** تحديث بيانات المؤسسة (CR, VAT)
