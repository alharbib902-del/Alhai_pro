## 🎨 Design System Usage

### Colors:
```dart
Primary: AlhaiColors.primary
Success: AlhaiColors.success
Error: AlhaiColors.error
Warning: AlhaiColors.warning
```

### Typography:
```dart
Headline: AlhaiTypography.headlineMedium
Body: AlhaiTypography.bodyMedium
Label: AlhaiTypography.labelSmall
```

### Components:
```dart
AlhaiButton(onTap: () {})
AlhaiCard()
AlhaiAppBar()
```

## 📱 Mobile Patterns

### Swipe Gestures:
```
← Swipe Left: Reject/Delete
→ Swipe Right: Approve/Save
↓ Pull Down: Refresh
↑ Swipe Up: Load more
```

### Bottom Navigation:
```
[🏪 Stores] [⚠️ Alerts] [📊 Dashboard] [⚙️ Settings]
```
