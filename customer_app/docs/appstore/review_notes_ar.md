# ملاحظات لمراجع المتجر — تطبيق العميل

> هذه الملاحظات تُرفق مع كل إصدار جديد لمساعدة مراجعي App Store / Play Store.

---

## App Store (Apple) — Review Notes

### حساب تجريبي للمراجعة

```
رقم الهاتف: [يُضاف قبل الإرسال]
رمز OTP: [يُضاف — أو استخدم حساب demo مُعد مسبقًا]
```

> **ملاحظة:** التطبيق يتطلب رقم هاتف سعودي للتسجيل. سنوفر حسابًا تجريبيًا جاهزًا.

### شرح استخدام الموقع

التطبيق يطلب إذن الموقع (`When In Use`) لـ:
1. عرض البقالات القريبة من المستخدم
2. تحديد عنوان التوصيل تلقائيًا
3. تتبّع التوصيل على الخريطة

**لا يعمل الموقع في الخلفية.** يُستخدم فقط أثناء استخدام التطبيق.

### خطوات اختبار التطبيق

1. **التسجيل:** افتح التطبيق → أدخل رقم الهاتف → أدخل رمز OTP
2. **تصفّح المنتجات:** الشاشة الرئيسية تعرض البقالات المتوفرة → اختر بقالة → تصفّح المنتجات
3. **إضافة للسلة:** اضغط على منتج → اضغط "أضف للسلة"
4. **الطلب:** اذهب للسلة → اختر عنوان التوصيل → اختر طريقة الدفع → أكد الطلب
5. **تتبّع الطلب:** بعد الطلب، تظهر شاشة التتبّع مع الخريطة
6. **الملف الشخصي:** اذهب للملف الشخصي → يمكنك تعديل البيانات أو حذف الحساب

### حذف الحساب

التطبيق يوفّر خاصية حذف الحساب من داخل التطبيق:
- الملف الشخصي → حذف الحساب → تأكيد
- يتم حذف جميع البيانات الشخصية خلال 30 يومًا
- يُحتفظ بسجلات المعاملات وفقًا للقانون السعودي

### ملاحظات إضافية

- التطبيق متاح في **المملكة العربية السعودية** فقط
- اللغة الأساسية: **العربية** (RTL)
- لا يحتوي على محتوى غير لائق
- لا يحتوي على مشتريات داخلية (in-app purchases)
- لا يحتوي على إعلانات
- لا يجمع بيانات أطفال (18+ فقط)

## Google Play — Review Notes

### Store Listing Checklist

- [ ] App title and description in Arabic
- [ ] Screenshots uploaded (6 minimum)
- [ ] Feature graphic (1024x500)
- [ ] Privacy policy URL active
- [ ] Data safety section completed
- [ ] Content rating questionnaire completed
- [ ] Target audience: 18+

### Background Location Declaration

**لا يستخدم تطبيق العميل الموقع في الخلفية.** يُستخدم فقط `ACCESS_FINE_LOCATION` أثناء استخدام التطبيق لعرض الخريطة وتحديد عنوان التوصيل.

### Sensitive Permissions Justification

| Permission | Justification |
|-----------|--------------|
| `ACCESS_FINE_LOCATION` | Required to show nearby stores and set delivery address on map |
| `INTERNET` | Required for API communication with Supabase backend |
