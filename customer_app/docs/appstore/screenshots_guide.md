## Screenshot Design Guidelines

1. **Language:** Arabic (primary), English (optional secondary set)
2. **Direction:** RTL layout
3. **Theme:** Light mode (default)
4. **Device frame:** Use device mockup frames (iPhone 15 Pro, Pixel 8)
5. **Captions:** Add text overlay above/below the screenshot
6. **Brand colors:** Use Alhai brand colors for captions
7. **No real user data:** Use demo/test data only
8. **Currency:** All prices in SAR (ر.س)

## Feature Graphic (Play Store)

- **Size:** 1024 x 500 px
- **Content:** Alhai logo + tagline "بقالة الحي للتوصيل السريع"
- **Background:** Brand gradient or solid color

## App Icon

- **Size:** 1024 x 1024 px (source)
- **Current:** Already configured in Flutter assets
- **Requirements:** No transparency (iOS), rounded corners applied automatically
