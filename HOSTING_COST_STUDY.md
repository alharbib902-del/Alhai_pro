## 💡 توصيات لتوفير التكاليف

### 1. ⬇️ تخفيض تكاليف OCR
- استخدام **Tesseract.js** للفواتير البسيطة (مجاني)
- استخدام Google Vision فقط للفواتير المعقدة
- **التوفير المتوقع**: 50-70%

### 2. ⬇️ تخفيض تكاليف الترجمة
- ترجمة المحتوى الثابت مرة واحدة وتخزينه
- استخدام ترجمة AI بدل API للمحتوى الديناميكي
- **التوفير المتوقع**: 80%

### 3. ⬇️ تخفيض تكاليف Maps
- استخدام **OpenStreetMap** (مجاني) للخرائط الأساسية
- استخدام Google فقط للـ Geocoding الدقيق
- **التوفير المتوقع**: 60%

### 4. ⬇️ استخدام Free Tiers

|--------|-------------|
| Google Cloud | $300 رصيد ابتدائي |
| Supabase | 500MB database |
