# منصة الحي - التوثيق الكامل
## منصة البقالة الذكية - نظام SaaS متكامل للبقالات في السعودية

---

## 🎯 نظرة عامة

**الحي** هي منصة SaaS شاملة متعددة المستأجرين للبقالات، تشمل:

- **7 تطبيقات** (الإدارة، نقطة البيع، العملاء، السائقين، الموزعين، المدير العام)
- **348 شاشة** إجمالي عبر جميع التطبيقات
- سوق **B2B للجملة**
- ميزات **الذكاء الاصطناعي**
- نظام **الدفع المقسم**
- **التتبع الفوري**

---

## ✨ الميزات الرئيسية

### 💰 نظام الجملة B2B
- سوق الموزعين
- طلبات بالجملة
- فوترة آلية
- رسوم المنصة (2%)

### 💳 الدفع المقسم
- نقد + بطاقات متعددة + آجل
- معاملة واحدة، طرق دفع متعددة
- إيصالات محسّنة

### 🤖 إعادة الطلب الذكية بالذكاء الاصطناعي
- تحليل ذكي للمخزون
- طلب استباقي
- اختيار أفضل سعر
- طلب بضغطة واحدة (من 50 دقيقة → 5 ثواني!)

### 📊 ميزات أخرى
- بنية متعددة المستأجرين
- تتبع فوري
- 20 نوع إشعار
- دعم 6 لغات

## 🚀 التقنيات المستخدمة

### الواجهة الأمامية
- Flutter 3.x (تطبيقات الموبايل)
- React/Next.js (تطبيقات الويب)
- alhai_design_system (مكونات UI)

### الخلفية
- Supabase (PostgreSQL، Auth، Realtime)
- Cloudflare R2 (التخزين)
- Edge Functions

### واجهات برمجة التطبيقات
- Google Maps SDK
- Cloud Translation API
- بوابات الدفع (مدى، فيزا، ماستركارد)
