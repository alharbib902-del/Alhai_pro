كيف تبدأ؟

إذا كنت مطورًا جديدًا في الفريق، اتبع هذه الخطوات للبدء بسرعة:

  1. استنسخ المستودع

    قم بنسخ المستودع من GitHub وتأكد من تثبيت Flutter SDK (الإصدار 3.24 أو أحدث)

  2. اقرأ البنية العامة

    ابدأ بصفحة البنية العامة لفهم هيكل المشروع والحزم المشتركة

  3. تعرف على قاعدة البيانات

    اطلع على مخطط قاعدة البيانات لفهم الجداول والعلاقات الأساسية

  4. راجع البنية المعمارية

    اقرأ البنية المعمارية لفهم أنماط التصميم وإدارة الحالة

  5. ابدأ التطوير

    اتبع دليل المطور لإعداد بيئة التطوير وتشغيل التطبيق محليًا
