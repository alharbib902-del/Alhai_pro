## فئات الاختبارات لكل شاشة

كل شاشة تُختبر في 8 فئات:

| الرمز | الفئة | الوصف |
|-------|-------|-------|
| **UI** | واجهة المستخدم | تحميل، عناصر مرئية، تخطيط |
| **RESP** | التجاوب | موبايل 375px، تابلت 768px، ديسكتوب 1920px |
| **RTL** | اتجاه عربي | نصوص، أيقونات، حقول بالاتجاه الصحيح |
| **DARK** | الوضع الداكن | ألوان، تباين، قابلية القراءة |
| **DB** | قاعدة البيانات | تحميل، حفظ، مزامنة |
| **ERR** | معالجة الأخطاء | رسائل خطأ، حالات فارغة، إعادة محاولة |
| **PERF** | الأداء | سرعة تحميل < 2 ثانية، سلاسة التمرير |
| **FUNC** | الوظائف | اختبارات خاصة بالشاشة |
