## Recommended Screenshots (6 screens)

### Screenshot 1: Orders Dashboard
- **Screen:** Main screen showing available/pending orders
- **Caption AR:** "استلم طلبات التوصيل"
- **Caption EN:** "Accept delivery orders"
- **Notes:** Show order cards with store name, distance, estimated earnings

### Screenshot 2: Order Details
- **Screen:** Single order detail with items and customer address
- **Caption AR:** "تفاصيل الطلب كاملة"
- **Caption EN:** "Complete order details"
- **Notes:** Show item list, delivery address (masked), payment method

### Screenshot 3: Navigation / Map
- **Screen:** In-delivery view with map and navigation
- **Caption AR:** "ملاحة مباشرة لعنوان العميل"
- **Caption EN:** "Direct navigation to customer"
- **Notes:** Show map with route, estimated time, delivery status

### Screenshot 4: Delivery Confirmation
- **Screen:** Proof of delivery / confirmation screen
- **Caption AR:** "أكّد التسليم بسهولة"
- **Caption EN:** "Confirm delivery easily"
- **Notes:** Show delivery confirmation UI, optional photo capture

### Screenshot 5: Earnings Dashboard
- **Screen:** Earnings summary with completed deliveries
- **Caption AR:** "تتبّع أرباحك"
- **Caption EN:** "Track your earnings"
- **Notes:** Show daily/weekly earnings, completed tasks count

### Screenshot 6: Profile
- **Screen:** Driver profile and settings
- **Caption AR:** "ملفك الشخصي"
- **Caption EN:** "Your profile"
- **Notes:** Show driver info, availability toggle, settings

## Design Guidelines

Same as customer_app:
- Arabic primary, RTL layout
- Light mode, device frames
- Demo data only, SAR currency
- Brand colors for captions
