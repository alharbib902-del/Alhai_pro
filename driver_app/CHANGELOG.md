# Changelog - Driver App

All notable changes to the Delivery Driver app.

Format based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [Unreleased]

### Added

## [1.0.0] - 2026-01-25

### Added
- Initial release: 18 screens
- Delivery queue with assigned orders
- Real-time GPS tracking and location sharing
- Order acceptance and status updates
- Delivery proof: photo and signature capture
- Navigation integration for route guidance
- Delivery history and earnings summary
- Profile and vehicle management
- Push notifications for new delivery assignments
- Multi-language (7 languages) and RTL support
